👥 User Management

ℹ️ Why User Management?

Are you a solo user? If you use SmartGallery alone, you can safely ignore this panel. Everything will work as usual.
Working with clients or a team? Use this to grant staff access to the Main Interface, or invite clients to the read-only Exhibition Portal.
Note: The main admin user is automatic and must be typed entirely in lowercase; its password is set via the --admin-pass startup parameter. For all configuration parameters, please refer to the GitHub repository.

Role Main Interface Exhibition Workflows & Downloads
Admin / MANAGER / STAFF Yes Yes Full Access. Can view and download original files with full JSON workflows.
FRIEND No Yes Partial. Workflows visible in metadata. Downloads original files.
USER / CUSTOMER / GUEST No Yes Restricted. Workflows strictly hidden. Downloads are clean/stripped.

ℹ️ Access Control Guide

Solo Users: You don't need this! If you use SmartGallery alone on your local machine, you can safely ignore this panel. User management is only required if you want to share your work with clients or collaborate with a team.

The admin Account
The root administrator is automatically generated. You set the password when launching the app using the --admin-pass parameter.

Roles & Permissions (ACL)

Role Main Interface Exhibition Workflows & Downloads
Admin / MANAGER / STAFF Yes Yes Full Access. Original files + JSON workflows.
FRIEND No Yes Partial. Workflows visible. Original files.
USER / CUSTOMER / GUEST No Yes Restricted. Workflows hidden. Clean/stripped files only.

Note on Customers: When a CUSTOMER downloads an image, all generation metadata (Prompts, ComfyUI nodes, EXIF) is completely stripped by the server before the transfer begins. Your creative process remains 100% private.
